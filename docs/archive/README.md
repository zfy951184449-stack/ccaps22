# Docs Archive

This directory keeps historical or generated reference material that may still be useful for audit, schema lookup, or backtracking old implementation context, but should not be treated as the active source of truth for the current MVP runtime.

## What Belongs Here

- generated schema snapshots that can lag current code cleanup
- aggregated document dumps that are useful for occasional lookup but too noisy for default reading
- HTML renderings or alternative generated formats when a canonical Markdown source already exists

## Archive Policy

- Keep when the artifact has audit, schema, or forensic value.
- Move here when it is still useful occasionally but should not stay on the primary docs path.
- Delete only when it is fully reproducible, has no unique historical value, and adds more noise than retrieval value.
